---

## 📌 What is RepoRadar?

RepoRadar is a web app that lets you search for trending GitHub repositories by any topic — machine learning, web design, healthcare, cybersecurity, and more. Filter results by today, this week, or this month and find the most starred repos in any field instantly.

No installs. No sign-up. Just open and search.

---

## ✨ Features

- 🔍 **Topic Search** — Search any keyword and get real matching repos from GitHub
- 📅 **Time Filters** — Filter by Today, This Week, or This Month
- ⭐ **Live Data** — Pulls real-time data directly from the GitHub REST API
- 💻 **Language Tags** — See what language each repo is built in
- 🔗 **Direct Links** — Click any repo to go straight to its GitHub page
- ⚡ **No Backend** — Runs entirely in the browser, zero setup needed

---

## 🚀 How to Use

1. Go to **[ompatel369.github.io/Reporadar](https://ompatel369.github.io/Reporadar)**
2. Type any topic in the search bar (e.g. `machine learning`, `web design`, `rust`)
3. Choose a time period — Today / This Week / This Month
4. Browse results and click any repo to open it on GitHub

## 💡 Why I Built This

GitHub's built-in trending page shows a generic list with no topic search. RepoRadar lets you search trending repos by any specific topic — useful for developers, students, and researchers who want to find what's popular in their field right now.
